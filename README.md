## Commit 컨벤션
```
<head>(subject)

<body>

<footer>
```
* head, body, footer은 서로 줄바꿈(Enter)로 구분을 한다.

|head|body|footer|
| :-----: | :-----: | :-----: |
| 제목은 명령어로 간단하게(ex: Add 22.12.30.(금)TIL 추가) | 내용은 TIL에 있는 내용(ex: Closure, Type Properties 내용 추가) | 명령어만 짧고 간단하게(ex: Add) |

### head와 footer 앞에 붙일 컨벤션
* Add - 파일 추가
* Edit - 파일 수정
* Delete - 파일 삭제
* Chore - 그 외 내용관련
